# UDParty (Love2D)

Esta aplicación escucha los comandos enviados por la aplicación Android de UDParty

## Requicitos

- love: para ejecutar la app de escritorio
- (opcional) lua: para ejecutar el receiver de consola

## Ejecución

### App Love2D

```
love .
```

### App cli

```
lua receiver_cli.lua
```
